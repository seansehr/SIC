To run these Kinect + Processing sketches you'll need install SimpleOpenNI
https://code.google.com/p/simple-openni/wiki/Installation

You'll also need to put the SimpleOpenNI Processing lib in the /libraries folder in your Sketchbook location (see Processing Preferences if you are not sure where that is.
https://code.google.com/p/simple-openni/downloads/list

You'll also need a Kinect compatible with your system.
